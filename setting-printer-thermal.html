Setting Printer
Note:
  • Type Koneksi Server Local adalah kondisi komputer printer dalam satu jaringan dengan server aplikasi antrian.
  • Type Koneksi Server Network adalah kondisi dimana komputer printer dengan server aplikasi antrian (Exp. Aplikasi antrian berada di hosting/vps/mengguankan ip public).
  • Jika Type koneksi server network dan anda menggunakan domain maka tuliskan nama domain anda tanpa mengguankan protocolnya (Exp. example.com)
  • Jika menggunakan virtualhost dengan setting port pada server maka Nama Service Server boleh di kosongkan atau diisi sesuai dengan nama service aplikasi antrian.
